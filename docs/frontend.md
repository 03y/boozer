# Frontend
* Two frontends: web and bot.


## Web
* Vanilla JS, HTML, CSS.
* No frameworks.

### Pages
* `/` - Leaderboard.
* `/items` - List of all items.
* `/login` - Login page.
* `/submit` - Submit a consumption.
* `/new_item` - Add a new item.
* `/profile` - User profile.
* `/legal` - Legal information.
* `/signup` - Signup page.

### Authentication
* `HttpOnly` cookie.

## Bot
* Discord bot.
* Written in Python.
* Uses `discord.py`.
* Not yet implemented.
